Order Analyzer:
A Python tool to calculate restaurant performance metrics from JSON data.

Usage:
Run the script by passing the path to your JSON file:

Bash:
python order_analysis.py orders.json
The script will generate an output.json file containing the processed report.

Key Metrics:
Average order value and hourly revenue, completion rate and daily order frequency, top and bottom performing restaurants by revenue.

Environment:
Built for Python3
